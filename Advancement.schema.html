Key Type Description Examples
type String not-null The statistic type
All statistic types can be seen here: https://github.com/battlegames/Battle-Hub/wiki/Statistics
Except "exp" which is not supported.
name String not-null The advancement's name
description List A nice description for this advancement
icon Material not-null The icon
inherit_progress boolean Should a player's progressions be inherited from
previous advancements with the same statistics type?
This should be enabled as players can continue what
they have achieved without doing from the beginning.
progression Map not-null Different period of this advancements
They can be known as 'levels'		 
progression:
  '0':
    amount: 1
    reward:
      exp: 5
      money: 8
  '1':
    amount: 10
    reward:
      exp: 60
      money: 100
  '2':
    amount: 20
    reward:
      exp: 150
