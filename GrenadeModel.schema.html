Key Type Description Examples
explosion_effect BattleEffect The explosion effect
delay_time long The delay time before the grenade is going to 'boom'
Set to 0 to disable this feature
velocity_multiplier double The multiplier applies to the directional velocity
explosion_power double The power of the explosion
Set to 0 to prevent the explosion
fire.block_radius int The radius that inner blocks get ignited
Set to 0 to prevent the fire (for blocks)
fire.mob_radius double The radius that inner mobs get ignited
Set to 0 to prevent the fire (for mobs)
fire.mob_ticks int How long do mobs get ignited
Set to 0 to prevent the fire (for mobs)
skin ItemSkin not-null Set the item's skin
name String Give a name for the item
