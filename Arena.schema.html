Key Type Description Examples
name String The name of this arena
mode String not-null The game mode
icon PreparedItem not-null The icon of this arena
game_options GameOptions not-null Game mode settings
max_time long The maximum playing time
max_players int The maximum number of players
allow_late_joins boolean Able to join a game even it has started
final_exp_formula String not-null The formula for calculating the final exp
- a: number of headshots
- b: number of kills
- c: number of deaths
- d: 1 if won or 0 if lost
final_money_formula String not-null The formula for calculating the final money
- a: number of headshots
- b: number of kills
- c: number of deaths
- d: 1 if won or 0 if lost
end_commands.winners List Commands to be executed by the console at the end
Placeholders can be used within the command with
values parsed from each winner
end_commands.losers List Commands to be executed by the console at the end
Placeholders can be used within the command with
values parsed from each loser
render_gui_on_death boolean Re-renders the GUI on death
This option can prevent players from reusing old items
bungeecord.enabled boolean Enable the Bungeecord support for this arena
Note: You must enable the Bungeecord support for
the whole plugin first (see general.yml)
bungeecord.remote_servers List List of remote servers
Remote servers are places where the game happens
end_firework BattleFirework The firework to be spawned when the game ends
end_delay long The delay time before the game actually ends
result_broadcast.won List The message to be sent to the winners
result_broadcast.lost List The message to be sent to the losers
rollback Rollback Rollback settings
empty_regions List
