WorldSettings

Key Type Description Examples
locale String not-null The locale file used for messages and texts
Available: en_us.yml, vi_vn.yml
date_format.long_form String not-null The format of date in long-form (contains day, month, year, etc)
Read more: Java Date Format
date_format.short_form.hours String not-null The format of date in short-form. The maximum time unit is hour.
Read more: Java Date Format
date_format.short_form.minutes String not-null The format of date in short-form. The maximum time unit is minute.
Read more: Java Date Format
date_format.short_form.seconds String not-null The format of date in short-form. The maximum time unit is second.
Read more: Java Date Format
level_system.exp_to_level_formula String not-null The formula for converting X exp to Y level
level_system.level_to_exp_formula String not-null The formula for converting X level to Y exp
default_scoreboard BattleScoreboard The default scoreboard settings
default_chat BattleChat The default chat settings
storage.type StorageType not-null The type of storage
Available: file, mysql
storage.file.data_path String The path to the directory where we should store data
storage.mysql.hostname String The IP (or host name) of MySQL server
storage.mysql.port int The port MySQL server listens to
storage.mysql.database String The MySQL database
storage.mysql.username String The MySQL user's name
storage.mysql.password String The MySQL user's password
storage.mysql.datasource_properties ConfigurationSection The external configuration (for HikariCP's data source)
misc.anti_death_drops boolean This option prevents player items and exp from dropping
misc.resource_pack.enabled boolean Enabled force resource pack on join
misc.resource_pack.custom_url String Custom resource pack URL
misc.heal_on_game_start boolean Enable this option to heal all players when game started
misc.heal_on_game_end boolean Enable this option to heal all players after end game
misc.default_speed.walk double The walking speed
-1 <= speed <= 1
misc.default_speed.fly double The flying speed
-1 <= speed <= 1
misc.region_partition_size int The maximum size for each region partition
misc.entity_track_min_distance double The minimum distance to ensure a tracked entity has moved
misc.blood_effect.enabled boolean Should we show the blood effects
misc.blood_effect.particles_ratio double The amount of blood particles will be spawned per each health point
misc.material_hardness Map The hardness of material
(Legacy and future materials are supported: that means you can even specify 1.17 material when running a 1.16 server)		 
material_hardness:
  _default_: 10
  '0':
    material:
      - air
      - cave_air
      - void_air
    value: 0
  '1':
    material:
      - leather_helmet
      - leather_chestplate
      - leather_leggings
      - leather_boots
    value: 1
misc.entity_hardness Map The base hardness of entities (without equipment) 
entity_hardness:
  _default_: 3
  '0':
    types:
      - player
      - skeleton
      - stray
      - spider
    value: 5
bungeecord.enabled boolean Should we enable the Bungeecord support?
bungeecord.lobby_servers List The lobby servers.
If the player can't connect to the first one, the second will be chosen, and so on
until there is no server left
bungeecord.reconnect_tries_per_server int Here you can specify how many times does the player can try to reconnect. If exceed,
the next server in the given list will be chosen.
bungeecord.connection_delay long The delayed time between two connections per player
control.gun_shoot MouseClick The type of mouse click for shooting gun
control.gun_zoom MouseClick The type of mouse click for zooming in (with scope attached to gun)
control.grenade_throw MouseClick The type of mouse click for throwing grenade
control.auto_reload_gun boolean Automatically reload guns ammunition when needed
in_game_economy.currency.name String The name of the currency that used in-game
in_game_economy.currency.format String The price format (contains the symbol and the cost)
in_game_economy.init_balance double The initial balance (given on starting a game)
join_sign.dynamic_lines List The text on join signs.
It will be updated frequently (based on the option `update_time`)
You can use placeholders for arena and game.
join_sign.update_time long Time for each update to the text on join signs (in ticks)
Minimum are 20 ticks; set to lower will disable the task
thus make the text always fixed.
backpack.sounds.on_add_success BattleSound Sound to be played when an item is put into the backpack successfully
backpack.sounds.on_add_failure BattleSound Sound to be played when a wrong item is put into the backpack
backpack.sounds.on_remove BattleSound Sound to be played when the player takes out item from backpack
radio.message_format String Radio message format
custom_items.medical_kit.health_bonus int Additional health give to the player
custom_items.medical_kit.use_sound BattleSound Sound to be played when using the medical kit
custom_items.adrenaline_shot.use_sound BattleSound Sound to be played when using the adrenaline shot
