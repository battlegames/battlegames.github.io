Key Type Description Examples
positions List not-null List of slots belong to this component 
positions: 1 -> 9; 1 -> 3
positions: 3 -> 5; 4
positions:
 - 1 ; 2
 - 1 ; 2 -> 4
item PreparedItem The item to set in the GUI
All slots belong to this component will have the same item
functions.on_init List The function that gets called when one slot of this component is initialized
Read more: https://github.com/battlegames/Battle-Hub/wiki/GUI		 
on_init:
 - Common:CopyCurrentSlot("window" "positionOfThisSlot")
functions.on_click List The function that gets called when one slot of this component is clicked
Read more: https://github.com/battlegames/Battle-Hub/wiki/GUI		 
on_click:
  - Common:CancelEvent() # Prevent steal items
functions.on_render List The function that gets called when one slot of this component is going to be rendered
Set $cancel_render to `true` to prevent this action; `false` is set by default
It is not recommended to do any modifications to the item on that slot, you would be better
to look at on_rendered, which is called after the rendering has done.
Read more: https://github.com/battlegames/Battle-Hub/wiki/GUI		 
on_render:
  - Cache:SetBool("_cancel_render_" true) ? _slot_ == 27
functions.on_rendered List The function that gets called when one slot of this component was rendered successfully
Read more: https://github.com/battlegames/Battle-Hub/wiki/GUI		 
on_rendered:
  - Common:SetItemFromData("window" "value" $_window_positionOfThisSlot "not-null")
pagination String Pagination that support multiple items to display in this component
Read more: https://github.com/battlegames/Battle-Hub/wiki/GUI
