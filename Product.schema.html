Key Type Description Examples
icon PreparedItem Product's icon
package_name String A nice name for the package icon.
This product is 'a package' when it gives player booster, perks, exp or items
(for items, requires the amount of two or more). You can read more
at the Market schema (option 'default_product_icon.package')
The icon for a package will be created automatically when no icon is set
package_material Material The material for the package icon
What is a package? Read above (option 'package_name')
currency CurrencyType The currency to be used
price double The cost of this product
in_game_only boolean Make this product only available during the game
reserved_game_modes List Make this product only available during certain game modes
This option only takes effect if in_game_only set to true
All game modes are non-case-sensitive
executions.perform_commands List The commands to be performed by the console later
You can use placeholders here; they are parsed
from the buyer's info
executions.give_perks List The perks to be given later
executions.give_boosters List The boosters to be given later
executions.give_items.vanilla Map The vanilla items to be given later
executions.give_items.battle Multimap The Battle items to be given later
executions.give_exp.vanilla int The vanilla exp to be given later
executions.give_exp.battle long The Battle exp to be given later
functions.on_purchase List Function to be called when a player purchases this product, and the transaction
has not been created (no money is taken, nothing is given to the player)
This function is very useful that allows you to change the cost. For example,
the cost can be reduced by half if the player has a specific rank. Or you can
make a 'tier cost' system that increase the cost each time the player purchases
so the first time is $10, then $20, $50, $100. etc
You can also reject the transaction by setting $forbidden to `true`
Variables: $price, $currency, $forbidden (modifiable)
functions.on_purchased List Function to be called when a player purchased this product successfully
Variables: $price, $currency (read-only)
